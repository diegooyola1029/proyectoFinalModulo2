# Mini Spotify en Java

Proyecto académico que simula el funcionamiento básico de Spotify utilizando **Java**, **Programación Orientada a Objetos** y un **menú textual en consola**.

---

## Funcionalidades

- Inicio y cierre de sesión
- Biblioteca musical con canciones
- Creación de playlists
- Agregar canciones a playlists
- Reproducción de canciones
- Listado del catálogo musical
- Menú interactivo por consola

---

## Conceptos aplicados

- Clases y objetos
- Herencia
- Interfaces
- Encapsulación
- Arreglos
- Métodos y constructores
- Menú interactivo con `Scanner`

---
